# Elegance
An ecommerce template for building online clothing stores.

## Features
- Responsive design for all devices
- Easy customization with CSS variables
- Built with modern web technologies
- Fast loading times with optimized images and code
- SEO-friendly with clean URLs and meta tags
- A simple Website builder tool for easy customization and deployment

## Installation
To install Elegance, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/imrany/elegance.git
   ```

2. Install dependencies:
   ```
   go mod tidy
   ```

3. Start the development server:
   ```
   go run main.go
   ```
